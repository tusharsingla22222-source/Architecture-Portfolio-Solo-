# Architecture Portfolio

A professional architecture portfolio website showcasing architectural projects, design philosophy, and professional experience.

## Overview

This portfolio website presents a curated selection of architectural projects and design work, built using modern web technologies and hosted on GitHub Pages. The site features a clean, responsive design that highlights visual content and provides an intuitive navigation experience.

## Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile devices
- **Project Gallery**: Showcases architectural projects with high-quality images and detailed descriptions
- **About Section**: Professional background, education, and design philosophy
- **Contact Form**: Easy way for potential clients and collaborators to get in touch
- **Smooth Navigation**: Intuitive user interface with seamless page transitions
- **Image Optimization**: Fast loading times with optimized media content

## Live Demo

Visit the live portfolio at: [`https://architect-portfolio-solo.netlify.app/`]

## Technologies Used

- **HTML5**: Structure and semantic markup
- **CSS3**: Styling and animations
- **JavaScript**: Interactive elements and dynamic functionality
- **GitHub Pages**: Hosting platform

## Getting Started

### Prerequisites

- A modern web browser
- Git installed on your local machine
- Basic knowledge of HTML/CSS/JavaScript (for customization)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/tusharsingla22222-source/Architecture-Portfolio-Solo-.git
```

2. Navigate to the project directory:
```bash
cd Architecture-Portfolio-Solo-
```

3. Open `index.html` in your browser to view locally:
```bash
open index.html
```

Or use a local server:
```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if http-server is installed)
npx http-server
```

Then visit `http://localhost:8000` in your browser.

## Deployment

This portfolio is automatically deployed via GitHub Pages. Any changes pushed to the main branch will be reflected on the live site.

### Setting up GitHub Pages

1. Go to your repository settings
2. Navigate to the "Pages" section
3. Select the main branch as the source
4. Save the settings
5. Your site will be available at `https://[username].github.io/Architecture-Portfolio-Solo-/`

## Customization

### Adding New Projects

1. Add project images to the `images/projects/` directory
2. Edit `projects.html` to include the new project details:
```html
<div class="project-card">
  <img src="images/projects/project-name.jpg" alt="Project Name">
  <h3>Project Title</h3>
  <p>Project description...</p>
</div>
```

### Updating Personal Information

- Modify the About section in `about.html`
- Update contact details in `contact.html`
- Edit the header/footer information across all pages

### Styling Changes

- Primary styles: `css/style.css`
- Responsive breakpoints: `css/responsive.css`
- Color scheme and typography can be adjusted in the CSS variables
